# Claude Fitness 🦀💪

ワークアウト記録＋ゲーミフィケーションアプリ。トレーニングするとカニのマスコット（Claude）が成長します。

## デモ

👉 https://nittohbs-dev.github.io/claude-fitness/

## 特徴

- **5部位のトレーニング記録**: 腕・脚・背中・胸・肩
- **カニの成長システム**: 鍛えた部位に応じてカニの各パーツが大きくなる
- **6段階レベル**: ひよっこ → ルーキー → セミプロ → プロ → エリート → レジェンド
- **ゴール設定**: 目標を設定して達成をお祝い
- **カレンダー表示**: トレーニング履歴を一覧

## 技術スタック

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Zustand（状態管理）
- Framer Motion（アニメーション）

## 開発

```bash
# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build
```
